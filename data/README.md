# Dataset Information

The SQLite database used in this project is not included in the repository due to GitHub file size limits.

The database contains:
- Normalized restaurant metadata
- Review statistics
- Image and ZIP-level reference tables
- Preprocessed features used for modeling

### How to Obtain the Database

You may:
- Recreate the database using the data ingestion and preprocessing scripts in `src/`, or
- Request access to the prepared database for academic or evaluation purposes.
